## Full Description

Countpluse is a devotional japa counter designed to help you stay focused in
your daily chanting practice.

Use the app as a guest or log in with OTP, set your mantra or counting target,
track your daily goal, and review progress from the summary screen. The app is
built to keep the counting experience simple, spiritual, and distraction-free.

Key features:

- Tap-based daily japa counting
- Voice-assisted counting support
- Daily summary and progress insights
- Custom mantra label and daily goal
- Profile photo and home wallpaper customization
- Guest mode and login flow with OTP verification

Countpluse is intended for personal devotional practice and progress tracking.

## Permissions Rationale

`RECORD_AUDIO`
- Used for the optional "Count as you speak" voice-counting feature.

`CAMERA`
- Used only when the user chooses to capture a profile photo or wallpaper from
  the camera.

`INTERNET`
- Used for login, registration, OTP verification, and server communication.
